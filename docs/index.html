Auto-Review API

Sample Node.js + TypeScript API with a full CI/CD pipeline (GitHub Actions → GHCR → SSH deploy).

Endpoints

  • GET /health live — returns service status, uptime and timestamp.

Local development

npm install
npm run dev

Run with Docker

docker run --rm -p 3000:3000 \
  ghcr.io/<owner>/<repo>:latest

CI/CD pipeline

  1. CI on every PR: lint, format-check, typecheck, test+coverage, build.
  2. Docker Publish on push to main and version tags: build & push image to GHCR.
  3. Deploy after a successful publish: SSH to server, pull image, restart container, health-check.
  4. Pages on changes to docs/: deploy this site.
